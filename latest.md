## 分集脚本

### 第1集《又下错站了》
- 旁白：我第三次在同一个站提前下车。雨刚落下来时，周予安把黑伞撑到我头顶，像这件事他已经做过很多次。
- 对白：林见微：你怎么又在这儿？ 周予安：你怎么又下错站？
- 分镜：
  1. 地铁口雨幕与鞋尖停住。
  2. 黑伞从画面右侧伸进来。
  3. 周予安半身站在雨里，肩头有水。
  4. 林见微第一人称抬头看他。
  5. 两人并肩走出站口。
  6. 伞面雨点近景。
- 结尾钩子：他说，你常走错的出口，一共有三个。

### 第2集《他都记得》
- 旁白：第二天唐果告诉我，周予安每次下班都会绕去那三个出口看一圈。不是顺路，是怕我认不对回家的方向。
- 对白：唐果：你以为他为什么总是晚三分钟？ 林见微：你们全组都知道？
- 分镜：
  1. 办公室早晨，咖啡蒸汽轻飘。
  2. 唐果凑近小声八卦。
  3. 周予安在远处低头画图。
  4. 林见微视线飘过去又收回。
  5. 打印机旁他耳尖微红。
  6. 桌面被放下一张手绘路线卡。
- 结尾钩子：路线卡最后一格写着，不会走就给我发消息。

### 第3集《便利店的热牛奶》
- 旁白：我没发消息，却在公司楼下便利店看见阿姨递来两袋热牛奶。她说，另一个人已经帮我选好不加冰的那种。
- 对白：便利店阿姨：你朋友眼光不错。 林见微：他不是我朋友。 周予安：那我先从热牛奶开始。
- 分镜：
  1. 便利店暖光照在玻璃门上。
  2. 热牛奶装进纸袋。
  3. 林见微接袋时微微愣住。
  4. 周予安从货架后走出来。
  5. 两人视线短暂停住。
  6. 阿姨笑着低头找零钱。
- 结尾钩子：纸袋底部贴着一枚白色耳机盒形状的小贴纸。

### 第4集《他记得我的忌口》
- 旁白：我不喝冰饮、不吃太甜，这些连我自己都常忘。可周予安把午餐备注写得比点单页面还认真。
- 对白：林见微：你记性这么好吗？ 周予安：不是记性好，是怕你胃疼。
- 分镜：
  1. 外卖袋放到桌面。
  2. 备注栏被镜头扫过但不显示具体字。
  3. 林见微抬头，明显愣住。
  4. 周予安说完后稍稍移开视线。
  5. 唐果在一旁忍笑。
  6. 林见微低头捏紧筷子。
- 结尾钩子：晚上项目组通知加班，偏偏外面又开始下雨。

### 第5集《停电那一分钟》
- 旁白：晚上十点，公司整层楼突然停电。黑下来的一瞬间，周予安先把手机电量开省电，再把亮着的屏幕递给我看导航。
- 对白：周予安：你先记出口，我送你到站。 林见微：你自己呢？ 周予安：我认路。
- 分镜：
  1. 办公区灯灭，全景变暗。
  2. 手机屏幕亮起，照出两人脸侧。
  3. 周予安把外套搭到椅背给她挡冷。
  4. 林见微握住手机，手指轻颤。
  5. 应急灯慢慢亮起。
  6. 两人并排走出办公室。
- 结尾钩子：电梯镜面里，我第一次发现他伞柄上挂着我的同款耳机盒扣。

### 第6集《伞柄里的纸条》
- 旁白：我借伞回家时，伞柄缝隙里滑出一张小纸条。上面写着一串时间和站名，全是我过去下错站的记录。
- 对白：林见微：你从什么时候开始记的？ 周予安：从你第一次把耳机盒落在站台那天。
- 分镜：
  1. 伞柄细节特写。
  2. 小纸条滑出落到手心。
  3. 林见微停在路灯下看字。
  4. 周予安在雨里站得很直。
  5. 两人对视，空气安静。
  6. 雨滴顺伞骨往下落。
- 结尾钩子：他说，那天你明明很想哭，却还是先把别人撞掉的文件捡了起来。

### 第7集《我不是顺路》
- 旁白：我问他是不是一直顺路。他摇头，像终于把藏了很久的话放出来。
- 对白：周予安：我不是顺路，我是想确认你有没有平安回家。 林见微：那你现在还想确认吗？ 周予安：想，但要先问你愿不愿意。
- 分镜：
  1. 雨停后的台阶，路面潮亮。
  2. 周予安把伞收半寸，保持距离。
  3. 林见微抬头看他，情绪稳定下来。
  4. 他说“愿不愿意”时轻停顿。
  5. 林见微把伞往他那边挪一点。
  6. 两人一起笑得很浅。
- 结尾钩子：我第一次主动说，明天下班你别晚三分钟了，我来等你。

### 第8集《换我等你》
- 旁白：第二天下班，我故意留在工位上多坐了半站的时间。不是考验他，是想让他知道，等人这件事可以轮到我。
- 对白：周予安：今天怎么没先走？ 林见微：怕你认错出口。
- 分镜：
  1. 办公室空了一半。
  2. 时钟指针走过下班时间。
  3. 周予安收拾电脑时抬头。
  4. 林见微拿起黑伞站到他面前。
  5. 唐果从后面探头起哄。
  6. 两人一起往门外走。
- 结尾钩子：唐果喊了一句，别只顾谈伞，耳机盒还没修好呢。

### 第9集《我误会了》
- 旁白：我看见周予安把我的耳机盒扣交给了别的女生，心里那点刚冒头的勇气立刻往下掉。
- 对白：林见微：原来你也会帮别人。 周予安：那是修扣子的师傅。你先别走。
- 分镜：
  1. 楼下小店门口，女生背影接过小物件。
  2. 林见微脚步停住又想转身。
  3. 周予安追出门外叫住她。
  4. 他摊开手心，耳机盒扣已缝好。
  5. 背面多了一小块浅蓝补丁。
  6. 林见微眼神慢慢放松。
- 结尾钩子：他说，这样以后就算你再走丢，我也认得出你。

### 第10集《半站之后》
- 旁白：后来下雨，我还是偶尔会认错站牌。只是每次回头，都能看见那把黑伞在不远不近的地方等我。
- 对白：周予安：今天还借你半站吗？ 林见微：不用了，今天我想把整段路都一起走完。
- 分镜：
  1. 地铁口外细雨重新落下。
  2. 黑伞撑开，两人站得更近一点。
  3. 林见微把伞柄握到中间。
  4. 周予安侧头看她笑。
  5. 两人沿街走远，暖光便利店在后方。
  6. 第二天桌上放着两份早餐袋。
- 结尾钩子：早餐袋里夹着一张新路线卡，终点写着：周末见。
